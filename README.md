# Portfolio

A personal portfolio website with a pixel-art theme, built to showcase my projects, skills, and certifications.

## Features

- 🎨 Pixel-art aesthetic with DotGothic16 / Outfit font pairing
- 🌗 Light/dark mode toggle with persistence
- 🪪 Draggable hanging ID card with spring-back physics and flip-to-reveal contact info
- 🛠️ Skill tags powered by Devicons
- 🖼️ Project modals with preview images
- 📜 Clickable certification cards

## Tech Stack

- HTML
- CSS
- JavaScript
